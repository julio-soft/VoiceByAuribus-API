# Environments

VoiceByAuribus API is available in multiple environments.

## Production

```
API: https://api.auribus.io
Auth: https://auth.auribus.io
```

## Staging

```
API: https://staging-api.auribus.io
Auth: https://staging-auth.auribus.io
```

## Rate Limits

- 100 requests per minute per API key
- 1000 requests per hour per API key

Contact support for higher limits.
